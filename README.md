This repository will be used for practicing AWS Services. 
Furthermore, it aims several steps to gain hands-on experience with AWS Services. 

Current Stage: 
1. Deploy the Backend and the Frontend on EC2 instances 

Stages: 
1. Deploy the backend and frontend on EC2 instances (Monolithic Architecture)
2. Deploy the backend on EC2 instances and the frontend on S3 (Decoupling Frontend & Backend)
3. Containerize the backend and frontend and deploy them on ECS (Transition to Microservices)
4. Deploy the containerized backend and frontend on EKS (Fully Managed Kubernetes Deployment)
